# Build or Buy Analysis Assistant

This application helps analyze user stories and technical specifications to determine whether to build or buy a solution.

## Features
- File upload support for user stories and technical specifications
- Automated capability extraction
- System description generation
- Build/Buy analysis with deep research
- Interactive research planning
- Detailed final report generation

## Usage
1. Upload your user stories and technical specifications file
2. Review the extracted capability and system description
3. Choose to proceed with build/buy analysis
4. Review and approve the research plan
5. Get detailed build/buy analysis report

## Environment Variables
The following environment variables are required:
- `OPENAI_API_KEY`: Your OpenAI API key
- `TAVILY_API_KEY`: Your Tavily API key
